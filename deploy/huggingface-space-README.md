# Kensho

AI multi-agent platform for food, travel and shopping — LangChain + LangGraph,
served as a single Docker image (FastAPI API + React SPA).

This is the Hugging Face Space entry point. Copy this file's frontmatter to the
Space repo's `README.md`; the `Dockerfile` at the repo root is built automatically.
See `DEPLOY.md` for the full setup (Supabase Postgres + pgvector + Space secrets).
